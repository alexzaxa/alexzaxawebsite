FAQ

Συχνές ερωτήσεις

Μπορείς να φτιάξεις ιστοσελίδα για μικρή τοπική επιχείρηση;

Ναι. Η σελίδα μπορεί να είναι απλή, καθαρή και φτιαγμένη για πελάτες που έρχονται από Google Maps, Instagram ή προτάσεις.

Πώς γίνεται η πληρωμή;

Η πληρωμή δεν γίνεται αυτόματα μέσα από το site. Πρώτα επιβεβαιώνω το πακέτο, τα extras και την τελική τιμή. Μετά μπορείς να πληρώσεις με PayPal payment link ή τραπεζική μεταφορά. Συνήθως προτείνεται 50% προκαταβολή για να ξεκινήσει το project και 50% όταν είναι έτοιμο για παράδοση.

Περιλαμβάνονται αλλαγές / διορθώσεις;

Ναι, περιλαμβάνονται βασικές διορθώσεις ώστε το αποτέλεσμα να είναι σωστό και καθαρό. Μεγάλες νέες αλλαγές, πολλές επιπλέον σελίδες ή νέα λειτουργικότητα συζητιούνται ξεχωριστά πριν προστεθούν.

Πόσο γρήγορα απαντάς;

Συνήθως απαντάω μέσα σε 24 ώρες. Αν μου στείλεις από την αρχή social links, υπάρχουσα ιστοσελίδα, φωτογραφίες ή menu, μπορώ να καταλάβω πιο γρήγορα τι χρειάζεται.

Μπορείς να βοηθήσεις με domain και hosting;

Ναι. Μπορώ να σε καθοδηγήσω για οικονομικές ή δωρεάν επιλογές hosting όπως GitHub Pages, Cloudflare Pages ή Netlify και για σύνδεση domain όταν είσαι έτοιμος.

Μπορεί να δουλέψει η σελίδα με Instagram και Google Maps;

Ναι. Η ιστοσελίδα μπορεί να έχει καθαρά links προς Instagram, Google Maps, email, φόρμα επικοινωνίας και όποιο άλλο κανάλι χρησιμοποιεί η επιχείρηση.

Μπορεί να γίνει redesign σε παλιά σελίδα;

Ναι. Μπορούμε να κρατήσουμε τις σωστές πληροφορίες και να βελτιώσουμε εμφάνιση, δομή, mobile εμπειρία και CTA ενότητες.

Πώς λειτουργούν τα referrals;

Αν κάποιος μου στείλει επιχείρηση που τελικά γίνει πληρωμένο project, μπορώ να δώσω ενδεικτικά 20€ bonus ή 20€ έκπτωση σε δικό του website. Το referral πρέπει να σταλεί από τη ξεχωριστή referral σελίδα ή να φαίνεται καθαρά στο μήνυμα και επιβεβαιώνεται χειροκίνητα.
